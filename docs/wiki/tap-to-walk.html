Tap to Walk

Tap to Walk gives you two shortcuts for moving your spoofed location with a single tap, without opening the app. Both are off by default. Turn them on in Settings → Menus → Tap to Walk.

Floating map quick walk

When this is on, tapping anywhere on the floating map immediately starts walking to that point. The confirmation sheet is skipped — the walk begins straight away.

Useful when you already know where you want to go and don't need to choose between Walk, Teleport, or Walk via roads every time.

Screen tap-to-walk overlay

This adds a crosshair button to the floating widget panel. Tap it once to activate a transparent layer that sits over whatever app you're currently using. Your next tap on the screen — anywhere — is treated as a walk destination. The overlay then disappears automatically.

To cancel without walking anywhere, tap the close button in the bottom-right corner.

Scale setting

The overlay converts where you tapped on screen into a real-world GPS coordinate using the Scale value (shown in meters per pixel). A higher number means each pixel on screen represents more distance on the ground — useful when the game map is zoomed out. A lower number is better for zoomed-in maps where precision matters.

If your taps are landing too far from where you intended, adjust the scale to match how zoomed in the game is. Zooming the game map out generally gives more accurate results because small positioning errors become less significant.

Important note

The overlay assumes the map you're tapping is oriented with north at the top. If the game rotates the map to follow your character, taps may not land in the expected direction. The Compass orientation feature below can correct for this automatically.

Some apps can detect when location is being mocked and may take action on the account. Use this feature only in apps where you understand and accept that risk.

Compass orientation

When this is on, locationjoystick takes a screenshot the moment you open the tap-to-walk overlay and looks for the red north arrow in the compass. It uses the arrow's direction to rotate the walk target so it lands in the right place even when the game map is turned.

Setup

  1. Go to Settings → Menus → Tap to Walk.
  2. Make sure Screen tap-to-walk overlay is turned on first.
  3. Under Compass orientation, tap Open Settings to open Android's Accessibility Settings.
  4. Find locationjoystick compass in the list and enable it.
  5. Return to locationjoystick — the status will update to Enabled.
  6. Turn on Detect compass orientation.

Compass region

By default the app looks for the compass in the top-right corner of the screen. If the game you use puts the compass somewhere else, expand Compass region (advanced) in Settings and adjust the horizontal position, vertical position, and radius sliders to match the compass location.

Important note

Accessibility services can be detected by some apps. If the game you use monitors for accessibility services and restricts features when one is active, turn off Detect compass orientation or disable the locationjoystick compass service in Android Accessibility Settings.
